# おじさん言語 (Ojisan Language)

おじさん構文でプログラミングができるインタープリタ言語です。C11で実装されています。

```
「🍺 オッハー❗ おじさん言語の世界へようこそダヨ😃🍻 」 オッハー❗
```

## 特徴

- 日本語 + 絵文字ベースの構文
- 変数、関数、クラス、配列、辞書をサポート
- 例外処理 (`ダイジョウブ...ナンテネ`)
- モジュールimport (`取り寄せてヨ😃`)
- HTTP通信 (WinHTTP)
- VSCode シンタックスハイライト拡張同梱

## ビルド

MinGW (gcc) が必要です。

```bash
make
```

## 実行

```bash
./ojisan examples/hello.ojs
```

実行できるファイルの拡張子は `.ojs` と `.oji` のみです。

## 構文例

### 変数宣言と出力

```
チョット聞いてヨ😃 名前チャンは 「おじさん」 ナンダ😘
名前チャン オッハー❗
```

### 関数定義

```
挨拶チャンのやり方教えるネ😘 名前チャン
  「こんにちは、」 と 名前チャン と 「！」 オッハー❗
やり方おしまい❗
挨拶チャンにオネガイ😃 「太郎」
```

### 条件分岐

```
チョット聞いてヨ😃 xチャンは 10 ナンダ😘
xチャン ッテサ😃 5 ヨリ大きい カナ😃
  「大きいヨ」 オッハー❗
ダッタラ コッチ😘
  「小さいヨ」 オッハー❗
オッケー❗マル😘
```

### ループ

```
チョット聞いてヨ😃 iチャンは 0 ナンダ😘
iチャン ッテサ😃 5 ヨリ小さい デショ😃
  iチャン オッハー❗
  iチャンは iチャン たす 1 ナンダ😘
繰り返すヨ😘
```

### HTTP通信

```
チョット聞いてヨ😃 結果チャンは 取ってきてネ😘チャンにオネガイ😃 「https://httpbin.org/get」 ナンダ😘
結果チャン オッハー❗
```
